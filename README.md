# AuthApp

1. Instalar los módulos de node
```
npm install
```

2. Clonar el ```.env.template``` y renombrarlo a ```.env```

3. Llenar las variables de entorno

4. Crear Angular Envs (opcional)
```
npm run envs
```

5. Para development ejecutar:
```
npm run start
```

6. Para producción ejecutar:
```
npm run build
```

7. Para levantar el proyecto ejecutar:
```
ng serve
```
